# LedgerLang
LedgerLang, is a Domain Specific Language (DSL) tailored for financial computations and data analysis. The project implements a complete compiler pipeline comprising four stages: Lexical Analysis, Syntax Analysis (Parsing), Intermediate Code Generation, and Target Code Generation. LedgerLang allows
users to declare financial variables such as accounts, rates, and portfolios; perform arithmetic and logical computations; and control program flow using conditional and iterative constructs. The compiler is implemented entirely in Python and produces both Three-Address Code (TAC) as an intermediate representation and executable Python code as the final target. The compiler also generates a numbered listing file interleaved with error messages, and continues analysis even in the presence of errors through robust error recovery mechanisms.
